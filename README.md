### Requirements
 - Python3
 - NodeJS

### Installation

 - `npm install`
 - `pip install -r proxy_server/requirements.txt`
 - `npm run build` (broken, run watch)
 - `npm run electron`

to develop run
 - `npm run watch`
 - `npm run electron`
